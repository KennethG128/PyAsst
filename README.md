# PyAsst
## 目录结构
- PyAsst\
  - README.md  
  - requirements.txt    #环境依赖清单
  - .gitignore          #GIT同步排除
  - log\                #日志
  - lib\                #全局通用脚本库
  - env\                #python虚拟环境
  - scr\                #脚本
  - conf\               #配置目录
  - bin\                #执行程序
  - test\               #测试脚本

## 文件命名规范
- 文件夹 驼峰
- py文件名 驼峰
- lib文件名 PyAsst+内容.py

## 代码规范
- 路径变量定义到字符'/'
- 变量 全小写 下换线区分单词
- 方法名 驼峰
